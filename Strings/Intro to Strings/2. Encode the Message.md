#### Hint : add extra space for given message to avoid list index out of range error. T.C = O(N),  S.C = O(2*length of message)
```
def encode(m):
    s = ''
    count = 1
    m += ' '
    n = len(m)
    for i in range(n-1):
        if m[i] == m[i+1]:
            count += 1
        else:
            s += m[i] + str(count)
            count = 1
    return s         
```
