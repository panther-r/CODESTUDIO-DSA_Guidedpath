#### HINT = Find ceiling of target's position , T.C = O(logN)

```
def bestInsertPos(arr, n, m):
    if n==0:
        return 0
    
    first = 0
    last = n-1
    if m > arr[last]:
        return n
    while first <= last:
        mid = (first + last)//2
        if m < arr[mid]:
            last = mid - 1
        elif m > arr[mid]:
            first = mid + 1
        else:
            return mid

    return first
    
```
