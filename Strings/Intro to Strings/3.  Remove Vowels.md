#### HINT : use set in python coz for searching it takes O(1) i.e, const time. T.C = O(N),S.C = O(<=N)
```
def removeVowels(inputS):
    # Write your code here.
    n = len(inputS)
    s = {'a','e','i','o','u',"A","E","I","O","U"}
    out = ''
    for i in range(n):
        if inputS[i] not in s:
            out += inputS[i]
        else:
            out += ''
    return out
```
